# logistic-cost-lib
Library for calculating logistic cost

## Cart Structure
All libs take cart as argument
```
[{
    quantity: 500,
    sku: {
        price: 2.9,
        pretaxPrice: 2.71,
        weighted: true,
        weight: 0.0630,
        kdw_mm: 338,
        kdh_mm: 7.0,
        kdl_mm: 208
    }
}]
```
